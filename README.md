# AI-Based College Voting System

This project is a secure online college voting system developed using face recognition technology. It ensures that only registered users can vote and prevents fake or duplicate voting.

---

## 🔹 Features
- User Registration with Face Capture  
- Face Verification Login  
- One Person One Vote System  
- Admin Dashboard  
- Election Time Control  
- Voter Status Tracking  

---

## 🔹 Technologies Used
- HTML, CSS, JavaScript  
- PHP  
- MySQL  
- face-api.js  
- XAMPP  

---

## 🔹 How It Works
1. User registers and captures face  
2. Face data is stored in database  
3. During login, face is verified  
4. If matched → user allowed to vote  

---

## 🔹 Security Features
- Password Hashing  
- Face Authentication  
- Session Management  
- Limited Attempts for Login  

---

## 🔹 Project Objective
To build a secure and transparent college voting system using AI-based face recognition.
